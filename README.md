# FMS - 자동화 시스템 관리 프로그램 UI 개선 프로젝트
오픈소스 기초 프로젝트, 자동화 시스템 관리 프로그램 UI 개선 프로젝트 Repo

# 설치 방법
front-end
```
$cd front-end
$npm install
```
package*.json 의존성 파일에 의한 자동 설치


back-end
```
$pip install -r requirements.txt
```

# 기능
- Vue 3 Composition API
- Vue Router를 이용한 페이지 라우팅
- Vue I18n을 이용한 다국어 지원
  - 한국어 (ko)
  - 영어 (en)
